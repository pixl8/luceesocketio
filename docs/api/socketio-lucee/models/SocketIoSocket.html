socketio-lucee.models

Class SocketIoSocket

lucee.Component
    socketio-lucee.models.SocketIoSocket

Represents a single socket connection. The socket object can be used to send direct messages, to join/leave rooms and to broadcast to other sockets within the namespace and optional rooms.
